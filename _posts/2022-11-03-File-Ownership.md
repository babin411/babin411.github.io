- `chown`: To change the user ownership

```
    Input: chown -c <user> <filename>
    Ouput: Changed the owner ship of the filename to the given user.
```

- `chgrp`: To change the group ownership
  
```
    Input: chgrp -c <group> <filename>
    Ouput: Changed the owner ship of the filename to the given user.
```
