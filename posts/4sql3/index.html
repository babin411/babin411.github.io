  • What counts with COUNT()

      • COUNT(*)

      • COUNT(1/0)
    • Non-NULL Values
      • COUNT(d.YR)
      • COUNT(NULLIF(d.YR, 1990))

  • Data in nature

2
3

          SomeDate
      FROM dbo.SomeTable

  • Downsampling Data

    1
2
3

          SELECT
          CAST(SomeDate AS DATE) AS SomeDate
      FROM dbo.SomeTable

  • Further Downsampling

    1
2
3

          SELECT
          DATEADD(HOUR, DATEDIFF(HOUR,0,SomeDate), 0) AS SomeDate
      FROM dbo.SomeTable

    DATEDIFF(HOUR,0,'2019-08-11 06:21:16') = 1,048,470

          SELECT
          t.Month,
          t.Day,
          SUM(t.Events) AS Events
      FROM TABLE
      GROUP BY
          t.Month,
          t.Day
      WITH ROLLUP
      ORDER BY
          t.Month,
          t.Day;
